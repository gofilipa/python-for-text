exploring bs4#

what is bs4?#

bs4 is short for BeautifulSoup4, a python package for parsing HTML data. bs4’s power comes from using python syntax to access and manipulate HTML elements. This means that it uses the python language and its syntax to get information from pages written in the web’s main computer lanugage, HTML.

# import the following libraries to use bs4

import requests # to make https requests
from bs4 import BeautifulSoup # our web scraping library
import lxml # a parser for working with html data

# save the data from the website as a "soup" object

site = requests.get('https://translegislation.com/bills/2023/passed')
html_code = site.content
soup = BeautifulSoup(html_code, 'lxml')

Using dot syntax, we can access the element contained within the soup object

# get title

soup.title

<title>2023 Passed anti-trans bills: Trans Legislation Tracker</title>

the soup object#

This word “object” in Python is something you’ll hear often. It means a collection of data and functions that can work on that data. You can think of it as a way of representing real world objects (like this web page) in a way that is organized and accessible, so you can work with that information, with Python.

Let’s take an initial look into what this beautiful soup object can do. It takes the HTML source, the specific HTML elelements or “tags,” and makes it possible for us to access those tags using python syntax.

text#

Let’s go a little deeper than the element. We can access just the text (getting rid of tags like <p> or <h3> from elements.

soup.title.text

'2023 Passed anti-trans bills: Trans Legislation Tracker'

Combine this with variables, and we can save just the text.

bill_title = soup.h3.text

bill_title

'AL HB261'

attributes#

In addition to text, we can also get the attributes. Attributes are additional data that is contained within the HTML tag, like href, which stands for hyperlink reference. This is the link’s URL. To access the attributes like href, we use the syntax: tag['attr'].

soup.h3['class']

['chakra-heading', 'css-1vygpf9']

link_location = soup.a['href']

link_location

inspecting our page#

Let’s navigate to our target website. Go to https://translegislation.com/. Scroll down until you see “Passed anti-trans bills” heading. Click on the red button that says “View 2023 Passed Bills”. (Alternatively, just navigate directly to the page here.

Once you’re on the page, right click on a bill title, any bill title, and select inspect element (or whatever option is closest to that phrase in your menu). The inspector will pop up and tell you everything you need to know about that element, including it’s HTML tag h3, which contains the a and any attributes, like class or href.

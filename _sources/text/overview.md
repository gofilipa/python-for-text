# workshop overview
## learning objectives
Welcome to text analysis! In this workshop we will build on the skills from the previous ones, [intro to Python](../intro/overview.md), and [web scraping](../scrape/overview.md), in order to find and explore textual patterns with Python. Although the previous workshops will certainly help you understand the text analysis content in this workshop, they are not required. This workshop is written to be useful for all beginner to intermediate programmers. 

The workshop introduces how to use Python to programmatically search for textual patterns. By the end of the workshop, students will have:
- a basic understanding of text analysis and the stages of a text analysis project
- knowledge of `spaCy` library for Natural Language Processing (NLP)
- familiarity with some popular tasks for NLP, like Named Entity Recognition (NER)
- practice with writing different rules for extracting patterns from text
- practice with reading and understanding Python syntax
  
## Google Colab
Like the previous workshops in the series, this workshop will continue to use [Google Colab](https://colab.research.google.com/), a browser-based tool for running Python code. Like Google Docs, Google Colab offers a cloud environment for authoring content. This means that you can use Colab without having to download and install Python. If you are new to Google Colab, please see [this tutorial](https://colab.research.google.com/?utm_source=scs-index) for specifics on how to run Python within the Google Colab environment.
